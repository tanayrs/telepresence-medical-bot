# Telepresence Robotic Arm for Remote Dermatological Monitoring of Infectious Wards

## How to Run

### Web Interface

In `web-interface/`, Run:
```
  python3 web-interface
```

### Motor Test

In `motor-control/`, Run:
```
  python3 joystick.py
```

### Camera Testing Server

In `camera-test/`, Run:
```
  python3 camera-server.py
```
